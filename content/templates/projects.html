PWeb static site generator

The PWeb static site generator is the Python program that generates this site.

It takes advantage of the Jinja2 template engine. Instead of forcing a site structure as some other generators, it interprets a JSON configuration.

A part from static pages, PWeb can build a tag based blog. Check the source code on the PWeb GitHub repository.
